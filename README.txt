
TEAM SCHEDULER PROJECT

Steps:

1. Install Node.js from https://nodejs.org
2. Open terminal in this folder
3. Run:

   npm install

4. Start the server:

   npm start

Server runs at:
http://localhost:3000

Endpoints:

POST /login
POST /create-meeting
POST /respond/:id
GET  /best-times/:id
